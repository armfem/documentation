               .. container:: level2

                  White Pages is a PHP application that allows users to
                  search and display data stored in an LDAP directory.

                  The application can be used on standard LDAPv3
                  directories and Active Directory, as all searched
                  attributes can be set in configuration.

                  It has the following features:

                  -  

                     .. container:: li

                        Quick search: a simple input in menu bar
                        searching on some classic attributes

                  -  

                     .. container:: li

                        Advanced search: a full form to search on
                        several attributes

                  -  

                     .. container:: li

                        Directory : display of all entries in a table
                        form

                  -  

                     .. container:: li

                        Gallery: display of all entries with their photo

                  -  

                     .. container:: li

                        Search and display groups and members

                  -  

                     .. container:: li

                        Export results as CSV

                  -  

                     .. container:: li

                        Export entry as vCard

                  .. container:: notetip

                     White Pages code is hosted on GitHub:
                     https://github.com/ltb-project/white-pages
